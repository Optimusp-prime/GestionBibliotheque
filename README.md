Pour importer la base de données :

1. Ouvrir SQL Server Management Studio.
2. Se connecter au serveur SQL Server.
3. Crée une nouvelle Query
4. Coller le code sql server et cliquer sur Execute.
5. La base GestionBibliothequedemo sera créée avec ses tables et ses données de test.
6. Vérifier dans config/database.php que la connexion utilise :
   - Base : GestionBibliothequedemo
   - Utilisateur : bibli_user
   - Mot de passe : BibliPassword123!
